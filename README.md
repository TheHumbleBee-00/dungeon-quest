# dungeon-quest
☠ DUNGEON QUEST — a full pixel art roguelike built from scratch! 🎮


👾 What's in the Game

6 hand-drawn pixel enemies — Slime, Bat, Skeleton, Goblin, Demon, Dragon (each gets harder per floor)
Procedurally generated dungeons — different every run
Items — Potions, Swords, Shields, Gold
Turn-based combat with level-up system
Real retro sound effects via Web Audio API
CRT scanline + vignette effect
Screen shake on damage, particle bursts on hits
High score saved to your browser
